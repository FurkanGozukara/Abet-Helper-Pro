    ABET Calculator - Overview

    Welcome to the ABET Calculator, a comprehensive tool for managing course outcomes, student performance, and ABET assessment.

    This application is designed to help faculty and administrators:

    • Define program and course outcomes aligned with ABET criteria
    • Track student performance on exams and course assessments
    • Calculate achievement levels for course and program outcomes
    • Generate reports for ABET accreditation purposes
    • Maintain historical data for continuous improvement

    Use the navigation on the left to explore specific areas of the documentation.

    Quick Tip: Start by adding a course and defining your program and course outcomes before importing students and creating exams.

    Getting Started

    First-time Setup

    Follow these steps to get started with the ABET Calculator:

    1. Define Program Outcomes

      Start by adding program outcomes that align with your program's educational objectives and ABET criteria.

      Go to Program Outcomes
    2. Create a Course

      Add a course with its code, name, and term information.

      Add Course
    3. Define Course Outcomes

      Create course outcomes for your course and link them to relevant program outcomes.

    4. Import Students

      Add students to your course by importing from a file or pasting a list.

    5. Add Exams and Questions

      Create exams and associate exam questions with course outcomes.

    6. Enter Student Scores

      Record student performance on each question.

    7. Calculate Results

      Generate and view achievement levels for course and program outcomes.

    After completing these steps, you'll have a fully functional ABET assessment system for your course.

    Managing Courses

    Course Management

    Courses are the main containers for organizing your ABET assessment data.

    Adding a Course
    1. Navigate to the Courses page
    2. Click "Add Course"
    3. Enter a unique course code (e.g., CSE101)
    4. Enter the course name and term
    5. Click "Save Course"
    Course Detail Page

    Each course has a detail page showing:

    • Course information
    • Course outcomes
    • Exams
    • Student list
    • Actions (calculate results, manage weights, etc.)
    Course Weights

    You can adjust the weight of each exam in the final course calculation. The total weight must equal 100%.

    Tip: You can use the "Duplicate Course" feature to create a copy of a course for a new term while preserving outcomes and exam structures.

    Program & Course Outcomes

    Working with Outcomes
    Program Outcomes

    Program outcomes are high-level educational objectives that align with ABET criteria. They typically apply to an entire program.

    • Add program outcomes with clear, measurable descriptions
    • Assign unique codes (e.g., PO1, PÇ1) for easy reference
    • Program outcomes can be linked to multiple course outcomes
    Course Outcomes

    Course outcomes are specific learning objectives for a particular course. They should support one or more program outcomes.

    • Add course outcomes with measurable descriptions
    • Link each course outcome to relevant program outcomes
    • Course outcomes are assessed through exam questions
    Important: Establishing clear relationships between program outcomes, course outcomes, and exam questions is essential for accurate ABET assessment.

    Frequently Asked Questions

    Achievement levels for course outcomes are calculated by:

    1. For each question linked to a course outcome, calculating the average student performance percentage
    2. Weighting these question scores based on their maximum points
    3. Combining weighted scores to get an overall achievement level for the course outcome

    Program outcome achievement levels are derived from the linked course outcomes, weighted by their relationship strength.

    Make-up exams are handled as follows:

    1. Create a new exam and mark it as a "make-up exam"
    2. Select which original exam it replaces
    3. The make-up exam will inherit the weight of the original exam
    4. When calculating results, the system will use the make-up score for students who took it, and the original exam score for others

    This ensures accurate assessment while accommodating students who need to take make-up exams.
